# Guide du Portail Administrateur Scolaire

## Vue d'ensemble

Le portail administrateur scolaire NovaConnect fournit des outils complets pour les administrateurs d'école afin de gérer tous les aspects des opérations scolaires, y compris les dossiers des étudiants, la gestion du personnel, les finances et la configuration du système.

## Prise en main

### Connexion

1. Accédez à l'URL NovaConnect de votre école
2. Entrez vos identifiants d'administrateur
3. Complétez l'authentification à deux facteurs si activée
4. Sélectionnez votre école (si accès multi-écoles)

### Vue d'ensemble du tableau de bord

Le tableau de bord administrateur affiche :
- **Métriques clés** : Nombre d'étudiants, nombre de personnel, résumé financier
- **Approbations en attente** : Notes, présence, inscriptions en attente d'approbation
- **Alertes** : Paiement en retard, problèmes de présence, notifications système
- **Actions rapides** : Tâches administratives courantes
- **Activité récente** : Dernières actions système et mises à jour

## Configuration de l'école

### Profil de l'école

Accédez à **Paramètres > Profil de l'école** pour gérer :
- **Informations de base** : Nom, adresse, coordonnées
- **Logo de l'école** : Téléchargez et gérez l'image de marque de l'école
- **Structure académique** : Niveaux, divisions, départements
- **Coordonnées** : Téléphone, email, site web
- **Réseaux sociaux** : Liens et handles

### Gestion de l'année académique

1. Accédez à **Paramètres > Années académiques**
2. Consultez les années académiques actuelles et passées
3. Créez une nouvelle année académique :
   - Nom de l'année (ex: "2024-2025")
   - Dates de début et de fin
   - Configuration des trimestres/semestres
4. Définissez comme active lorsque prête
5. Archivez les anciennes années

### Configuration des notes

1. Accédez à **Paramètres > Notes**
2. Configurez :
   - **Échelles de notation** : Définissez les notes littérales et plages de pourcentage
   - **Calcul de la moyenne** : Définissez les pondérations et la méthode de calcul
   - **Notes de passage** : Pourcentage minimum de réussite
   - **Catégories de notes** : Pondérations des devoirs, tests, examens, projets
3. Enregistrez les politiques de notation

### Configuration des frais

1. Accédez à **Finance > Configuration des frais**
2. Définissez les types de frais :
   - Frais de scolarité
   - Frais d'inscription
   - Frais de matériel
   - Frais d'activités
3. Définissez les montants par niveau
4. Configurez les échéanciers de paiement :
   - Paiements uniques
   - Plans de versement
   - Dates d'échéance
5. Appliquez les remises et exemptions

## Gestion des utilisateurs

### Gestion du personnel

1. Accédez à **Utilisateurs > Personnel**
2. Cliquez sur **Ajouter un membre du personnel**
3. Remplissez les détails du personnel :
   - Informations personnelles
   - Rôle (Enseignant, Comptable, Administrateur, etc.)
   - Matières/Départements
   - Coordonnées
4. Définissez les permissions du compte
5. Envoyez l'invitation au compte

**Rôles disponibles** :
- **Super administrateur** : Accès complet au système
- **Administrateur scolaire** : Administration au niveau de l'école
- **Enseignant** : Gestion des classes et matières
- **Comptable** : Gestion financière
- **Support** : Accès limité pour les tâches de support

### Gestion des étudiants

#### Inscription de nouveaux étudiants

1. Accédez à **Utilisateurs > Étudiants**
2. Cliquez sur **Inscrire un étudiant**
3. Remplissez les informations de l'étudiant :
   - Détails personnels
   - Informations du parent/tuteur
   - Dossiers scolaires précédents
   - Informations médicales
4. Assignez :
   - ID d'étudiant/Matricule
   - Niveau scolaire
   - Classe/section
   - Campus
5. Générez le compte étudiant
6. Imprimez la confirmation d'inscription

#### Gestion des dossiers d'étudiants

1. Sélectionnez l'étudiant dans **Utilisateurs > Étudiants**
2. Consultez/modifiez :
   - Informations du profil
   - Historique académique
   - Dossiers de présence
   - Historique des paiements
   - Dossiers disciplinaires
3. Mettez à jour le statut :
   - Actif/Inactif
   - Transfert/Retrait
   - Diplômé

#### Importation en masse d'étudiants

1. Accédez à **Utilisateurs > Étudiants > Importer**
2. Téléchargez le modèle d'importation
3. Remplissez les données des étudiants
4. Téléversez le fichier rempli
5. Mappez les colonnes
6. Validez et importez

### Gestion des comptes parents

1. Accédez à **Utilisateurs > Parents**
2. Consultez tous les comptes parents
3. Liez les parents aux étudiants
4. Envoyez des invitations de compte
5. Réinitialisez les mots de passe parents
6. Mettez à jour les coordonnées

## Gestion académique

### Gestion des classes

#### Création de classes

1. Accédez à **Académique > Classes**
2. Cliquez sur **Créer une classe**
3. Configurez :
   - Nom de la classe (ex: "6A", "Form 1B")
   - Niveau scolaire
   - Année académique
   - Campus/Salle
   - Limite de capacité
4. Assignez un professeur principal
5. Ajoutez les étudiants inscrits

#### Assignations des enseignants

1. Accédez à **Académique > Assignations des enseignants**
2. Sélectionnez l'enseignant
3. Assignez :
   - Classes
   - Matières
   - Périodes
4. Consultez la charge d'enseignement
5. Gérez les substitutions

### Gestion des matières et curriculum

1. Accédez à **Académique > Matières**
2. Créez/modifiez des matières :
   - Nom et code de la matière
   - Description
   - Niveaux scolaires
   - Prérequis
3. Organisez par département
4. Assignez des responsables de matière

### Gestion de la présence

#### Consultation des rapports de présence

1. Accédez à **Académique > Présence**
2. Sélectionnez une plage de dates
3. Filtrez par :
   - Classe
   - Niveau scolaire
   - Étudiant
4. Consultez les statistiques :
   - Pourcentage de présence
   - Absentéisme chronique
   - Tendances et patterns

#### Approbations de présence

1. Accédez à **Académique > Présence > Approbations**
2. Examinez la présence soumise
3. Approuvez ou demandez des corrections
4. Ajoutez des remarques si nécessaire

### Gestion et approbation des notes

1. Accédez à **Académique > Notes > Approbations**
2. Consultez les notes brouillon soumises par les enseignants
3. Examinez pour vérifier l'exactitude
4. Approuvez ou renvoyez pour révision
5. Les notes publiées sont visibles par les étudiants/parents

### Génération des bulletins

1. Accédez à **Académique > Bulletins**
2. Sélectionnez l'année académique et le trimestre
3. Choisissez une classe ou un étudiant individuel
4. Générez les bulletins
5. Prévisualisez et :
   - Imprimez individuellement
   - Imprimez en masse pour la classe
   - Envoyez par email aux parents
   - Téléchargez en PDF

### Gestion des examens

#### Création de sessions d'examens

1. Accédez à **Académique > Examens**
2. Cliquez sur **Créer une session d'examens**
3. Configurez :
   - Nom de l'examen (ex: "Examens de mi-session 2024")
   - Plage de dates
   - Types d'examens
4. Ajoutez l'horaire des examens :
   - Matière
   - Date et heure
   - Durée
   - Attribution des salles
   - Surveillants
5. Publiez l'horaire des examens

#### Gestion des notes d'examens

1. Accédez à **Académique > Examens > [Sélectionner l'examen]**
2. Consultez l'horaire des examens
3. Surveillez la progression de la saisie des notes
4. Examinez les notes soumises
5. Approuvez pour publication

## Gestion financière

### Gestion des échéanciers de frais

1. Accédez à **Finance > Échéanciers de frais**
2. Consultez tous les échéanciers de frais
3. Créez un nouvel échéancier :
   - Sélectionnez l'année académique
   - Choisissez le type de frais
   - Définissez les dates d'échéance
   - Assignez aux niveaux scolaires
4. Appliquez à des étudiants individuels ou en masse

### Suivi des paiements

1. Accédez à **Finance > Paiements**
2. Consultez tous les paiements :
   - Filtrez par date, étudiant, statut
   - Recherchez par référence de paiement
   - Exportez les rapports de paiement
3. Actions de paiement :
   - Enregistrez un paiement manuel
   - Générez des reçus
   - Traitez les remboursements
   - Mettez à jour le statut de paiement

### Rappels de paiement

1. Accédez à **Finance > Rappels de paiement**
2. Configurez les règles de rappel :
   - Avant la date d'échéance (7, 3, 1 jours)
   - Après la date d'échéance (1, 7, 14 jours)
3. Sélectionnez le mode de livraison :
   - Email
   - SMS
   - Notification push
4. Envoyez un rappel test
5. Activez les rappels automatiques

### Rapports financiers

Générez des rapports financiers :
1. Accédez à **Finance > Rapports**
2. Sélectionnez le type de rapport :
   - **Rapport d'encaissement** : Total encaissé vs impayé
   - **Rapport de vieillissement** : Paiements en retard par jours
   - **Historique des paiements** : Journal des transactions
   - **Rapport des exonérations** : Exemptions et remises
3. Définissez la plage de dates et les filtres
4. Générez et exportez (PDF, Excel)

### Gestion de la paie (si activé)

1. Accédez à **Finance > Paie**
2. Gérez les salaires du personnel :
   - Configurez les grilles salariales
   - Enregistrez les paiements
   - Générez les fiches de paie
   - Suivez les déductions et avantages

## Communication et annonces

### Annonces de l'école

1. Accédez à **Communication > Annonces**
2. Cliquez sur **Créer une annonce**
3. Remplissez :
   - Titre et contenu
   - Public cible (Tous, Étudiants, Parents, Personnel)
   - Niveau de priorité
   - Pièces jointes
4. Planifiez ou envoyez immédiatement
5. Suivez les accusés de lecture

### Centre de messages

1. Accédez à **Communication > Messages**
2. Consultez tous les messages de l'école
3. Répondez aux demandes
4. Déléguez au personnel approprié
5. Surveillez les temps de réponse

### Diffusions SMS et Email

1. Accédez à **Communication > Diffusions**
2. Créez un message de diffusion
3. Sélectionnez les destinataires :
   - Tous les parents
   - Niveaux scolaires spécifiques
   - Groupe personnalisé
4. Rédigez le message
5. Envoyez immédiatement ou planifiez

## Rapports et analytiques

### Rapports académiques

1. Accédez à **Rapports > Académique**
2. Générez :
   - **Rapports de performance** : Moyennes de classe/matière
   - **Distribution des notes** : Taux de réussite/échec
   - **Rapports de présence** : Par classe, niveau, étudiant
   - **Rapports comparatifs** : Comparaison d'une année à l'autre
3. Exportez et partagez

### Rapports d'inscription

1. Accédez à **Rapports > Inscription**
2. Consultez :
   - Inscription actuelle par niveau
   - Tendances d'inscription
   - Statistiques d'abandon/transfert
   - Utilisation de la capacité

### Rapports du personnel

1. Accédez à **Rapports > Personnel**
2. Générez :
   - Annuaire du personnel
   - Analyse de la charge d'enseignement
   - Dossiers de présence
   - Résumés de paie

### Rapports personnalisés

1. Accédez à **Rapports > Personnalisé**
2. Sélectionnez les sources de données
3. Définissez les filtres et paramètres
4. Enregistrez le modèle de rapport pour utilisation future

## Administration système

### Contrôle d'accès utilisateur

1. Accédez à **Paramètres > Permissions**
2. Gérez l'accès basé sur les rôles :
   - Consultez les rôles existants
   - Créez des rôles personnalisés
   - Assignez des permissions aux rôles
3. Audit de l'accès utilisateur :
   - Consultez l'historique de connexion
   - Surveillez les tentatives échouées
   - Examinez les changements de permissions

### Importation et exportation de données

#### Importer des données

1. Accédez à **Paramètres > Données > Importer**
2. Importations prises en charge :
   - Étudiants (inscription en masse)
   - Personnel
   - Notes
   - Échéanciers de frais
3. Téléchargez les modèles
4. Téléversez les fichiers
5. Validez et importez

#### Exporter des données

1. Accédez à **Paramètres > Données > Exporter**
2. Sélectionnez le type de données :
   - Dossiers d'étudiants
   - Données académiques
   - Données financières
   - Journaux système
3. Définissez les filtres
4. Exportez (CSV, Excel, PDF)

### Sauvegarde et récupération

1. Accédez à **Paramètres > Sauvegarde**
2. Configurez les sauvegardes automatisées :
   - Heure de sauvegarde quotidienne
   - Période de rétention
   - Emplacement de sauvegarde
3. Sauvegarde manuelle :
   - Créez une sauvegarde à la demande
   - Téléchargez le fichier de sauvegarde
4. Récupération :
   - Sélectionnez le fichier de sauvegarde
   - Prévisualisez le contenu
   - Exécutez la récupération

### Journaux système

1. Accédez à **Paramètres > Journaux système**
2. Consultez les journaux par catégorie :
   - Activité utilisateur
   - Modifications de données
   - Erreurs système
   - Événements de sécurité
3. Recherchez et filtrez les journaux
4. Exportez pour audit

## Paramètres et configuration

### Paramètres généraux

1. Accédez à **Paramètres > Général**
2. Configurez :
   - **Format de date/heure** : Préférences régionales
   - **Langue** : Langue du système
   - **Devise** : Transactions financières
   - **Fuseau horaire** : Calendrier académique

### Paramètres de notification

1. Accédez à **Paramètres > Notifications**
2. Configurez les notifications système :
   - Paramètres du serveur email
   - Configuration de la passerelle SMS
   - Paramètres de notification push
3. Définissez les règles de notification :
   - Publication des notes
   - Rappels de paiement
   - Alertes de présence
   - Annonces système

### Paramètres de sécurité

1. Accédez à **Paramètres > Sécurité**
2. Configurez :
   - **Stratégie de mot de passe** : Complexité, expiration
   - **Délai d'expiration de session** : Durée de déconnexion automatique
   - **Authentification à deux facteurs** : Activer/désactiver
   - **Liste blanche IP** : Restreindre l'accès par IP
   - **Journal d'audit** : Suivre toutes les actions

### Paramètres d'intégration

1. Accédez à **Paramètres > Intégrations**
2. Configurez les intégrations tierces :
   - Passerelles de paiement
   - Services email
   - Fournisseurs SMS
   - Logiciels comptables
   - Systèmes de gestion d'apprentissage

## Gestion de l'application mobile

### Configuration de l'application enseignant

1. Accédez à **Paramètres > Applications mobiles**
2. Configurez les fonctionnalités de l'application enseignant :
   - Paramètres de synchronisation de données hors ligne
   - Suivi de présence
   - Permissions de saisie des notes
   - Fonctionnalités de communication

### Applications parents et étudiants

1. Accédez à **Paramètres > Applications mobiles > Étudiant/Parent**
2. Configurez :
   - Permissions d'accès
   - Paramètres de notification
   - Fonctionnalités disponibles
   - Version de l'application personnalisée

## Bonnes pratiques

### Opérations quotidiennes
- Consultez les alertes du tableau de bord quotidiennement
- Approuvez les notes et la présence en attente
- Répondez aux messages rapidement
- Sauvegardez les données avant les changements majeurs

### Tâches hebdomadaires
- Examinez les rapports de présence
- Vérifiez le statut des encaissements
- Envoyez les rappels de paiement
- Examinez les journaux système pour les problèmes

### Tâches mensuelles
- Générez les rapports financiers
- Examinez les changements d'inscription
- Mettez à jour l'accès utilisateur si nécessaire
- Examinez et optimisez les performances du système

### Fin de trimestre/semestre
- Publiez toutes les notes
- Générez les bulletins
- Archivez les dossiers académiques
- Préparez pour le prochain trimestre

### Sécurité
- Examinez régulièrement l'accès utilisateur
- Surveillez les activités suspectes
- Gardez le logiciel à jour
- Effectuez des audits de sécurité

## Dépannage

### Problèmes courants

**Performance lente du système**
- Vérifiez le nombre d'utilisateurs simultanés
- Examinez la taille de la base de données et exécutez la maintenance
- Videz le cache système
- Contactez le support si le problème persiste

**Les utilisateurs ne peuvent pas se connecter**
- Vérifiez que les identifiants sont corrects
- Vérifiez si le compte est verrouillé
- Réinitialisez le mot de passe si nécessaire
- Examinez les paramètres d'authentification

**Les notes n'apparaissent pas**
- Vérifiez si les notes sont publiées
- Vérifiez le workflow d'approbation
- Examinez les permissions de l'enseignant
- Vérifiez l'horaire de publication des notes

**Écarts de paiement**
- Réconciliez les dossiers de paiement
- Vérifiez les entrées en double
- Examinez la configuration des échéanciers de frais
- Auditez l'historique des paiements

### Obtenir de l'aide

- **Documentation système** : Disponible dans **Aide > Documentation**
- **Support technique** : support@novaconnect.com
- **Gestionnaire de compte** : Votre gestionnaire de compte dédié
- **Forum communautaire** : community.novaconnect.com
- **Webinaires de formation** : Sessions mensuelles programmées

## Conformité et audit

### Confidentialité des données
- Conformité au RGPD/lois locales de protection des données
- Mise en œuvre de politiques de rétention des données
- Sécurisez les informations personnelles
- Fournissez l'accès aux données/suppression sur demande

### Traçabilité d'audit
- Toutes les actions sont journalisées automatiquement
- Examen régulier des journaux système
- Exportez les rapports d'audit pour conformité
- Enquêtez sur les activités suspectes

### Exigences de sauvegarde
- Sauvegardes automatisées quotidiennes
- Vérification hebdomadaire des sauvegardes
- Stockage de sauvegarde hors site mensuel
- Test annuel de récupération après sinistre

## Gestion multi-écoles

(Pour les super administrateurs gérant plusieurs écoles)

### Changement d'école
- Basculez entre les écoles sans reconnexion
- Consultez les tableaux de bord agrégés
- Comparez les performances entre les écoles

### Rapports inter-écoles
- Générez des rapports consolidés
- Agréguez les données financières
- Comparez les tendances d'inscription
- Partagez les meilleures pratiques

### Gestion des licences
- Consultez l'utilisation des licences par école
- Mettez à niveau ou réduisez les plans
- Gérez les abonnements scolaires
- Suivez la facturation entre les écoles
