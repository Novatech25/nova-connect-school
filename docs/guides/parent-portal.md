# Guide du Portail Parents

## Vue d'ensemble

Le Portail Parents NovaConnect permet aux parents de surveiller les progrès académiques de leurs enfants, leur assiduité, leurs dossiers financiers, et de communiquer avec les enseignants.

## Prise en main

### Connexion

1. Accédez à l'URL NovaConnect de votre école
2. Entrez votre email/téléphone et votre mot de passe
3. Cliquez sur "Se connecter"

### Configuration du compte

Les nouveaux utilisateurs doivent :
1. Vérifier leur adresse email
2. Lier les enfants au compte en utilisant leur matricule
3. Configurer les préférences de notification
4. Compléter les informations du profil

## Tableau de bord

Le tableau de bord parent affiche :
- **Vue d'ensemble des enfants** : Accès rapide aux informations de chaque enfant
- **Activité récente** : Dernières notes, assiduité et annonces
- **Événements à venir** : Examens, vacances, réunions parents-professeurs
- **Alertes de paiement** : Échéances et paiements en retard

## Gestion de plusieurs enfants

### Bascule entre les enfants

1. Utilisez le sélecteur d'enfant en haut de la page
2. Sélectionnez l'enfant que vous souhaitez visualiser
3. Toutes les pages se mettent à jour pour afficher les données de l'enfant sélectionné

### Vue combinée

Certaines pages affichent tous les enfants ensemble :
- **Paiements** : Les frais de tous les enfants dans une seule vue
- **Annonces** : Annonces à l'échelle de l'école
- **Messages** : Toutes les communications dans une seule boîte de réception

## Suivi académique

### Consultation des notes

1. Sélectionnez l'enfant dans le menu déroulant
2. Naviguez vers **Académique > Notes**
3. Consultez les notes publiées par trimestre
4. Cliquez sur une note pour voir le détail

**Note** : Seules les notes publiées sont visibles. Les brouillons de notes restent cachés jusqu'à leur publication par les enseignants.

### Bulletins

1. Sélectionnez l'enfant dans le menu déroulant
2. Naviguez vers **Académique > Bulletins**
3. Sélectionnez l'année scolaire et le trimestre
4. Téléchargez le PDF pour l'impression ou l'enregistrement

### Suivi de l'assiduité

1. Sélectionnez l'enfant dans le menu déroulant
2. Naviguez vers **Académique > Assiduité**
3. Consultez :
   - Les enregistrements d'assiduité quotidiens
   - Les statistiques d'assiduité (% de présence)
   - Les modèles d'absence et les alertes
   - Les remarques des enseignants

### Emploi du temps

1. Sélectionnez l'enfant dans le menu déroulant
2. Naviguez vers **Académique > Emploi du temps**
3. Consultez l'horaire hebdomadaire
4. Voyez les informations de contact des enseignants

## Gestion financière

### Consultation des frais

1. Naviguez vers **Finance > Frais**
2. Sélectionnez "Tous les enfants" ou un enfant individuel
3. Consultez :
   - Les soldes impayés
   - Le détail des frais (scolarité, matériels, etc.)
   - Les dates d'échéance
   - Le statut des paiements

### Historique des paiements

1. Naviguez vers **Finance > Paiements**
2. Filtrez par enfant si nécessaire
3. Consultez toutes les transactions
4. Téléchargez les reçus

### Effectuer des paiements

Les options de paiement dépendent de la configuration de l'école :
- **Paiement en ligne** : Carte de crédit, virement bancaire
- **Mobile Money** : Plateformes de paiement mobile
- **En personne** : Payer au bureau de l'école

Pour effectuer un paiement :
1. Naviguez vers **Finance > Effectuer un paiement**
2. Sélectionnez l'enfant et le type de frais
3. Entrez le montant
4. Choisissez le mode de paiement
5. Confirmez et traitez

### Rappels de paiement

Configurez des rappels automatiques :
1. Naviguez vers **Paramètres > Notifications**
2. Activez les rappels de paiement
3. Choisissez le timing des rappels (7 jours, 3 jours avant la date d'échéance)
4. Sélectionnez le mode de notification (email, SMS, push)

## Communication

### Messagerie avec les enseignants

1. Naviguez vers **Messages > Composer**
2. Sélectionnez le destinataire (enseignant, administrateur)
3. Sélectionnez l'enfant si pertinent
4. Rédigez le message et envoyez

### Réunions parents-professeurs

1. Naviguez vers **Réunions**
2. Consultez les réunions planifiées
3. Demandez une nouvelle réunion
4. Ajoutez à l'agenda

### Annonces de l'école

1. Naviguez vers **Annonces**
2. Consultez par :
   - Toutes les annonces
   - Par enfant
   - Par catégorie (académique, événements, urgence)

### Gestion des notifications

Configurez dans **Paramètres > Notifications** :
- Notifications par email
- Notifications par SMS
- Notifications push
- Fréquence (immédiat, résumé quotidien)

## Documents et dossiers

### Accès aux documents des enfants

1. Sélectionnez l'enfant dans le menu déroulant
2. Naviguez vers **Documents**
3. Consultez et téléchargez :
   - Les bulletins
   - Les cartes d'identité
   - Les certificats
   - Les documents d'inscription

### Téléchargement de documents

1. Naviguez vers **Documents > Télécharger**
2. Sélectionnez l'enfant
3. Choisissez le type de document
4. Téléchargez le fichier (PDF, JPG, PNG)

## Dépannage

### Problèmes courants

**Enfant n'apparaît pas dans le compte**
- Vérifiez que le matricule de l'enfant est correct
- Contactez l'administration de l'école pour lier le compte
- Vérifiez que l'enfant est inscrit pour l'année scolaire en cours

**Notes non visibles**
- Confirmez que les notes sont publiées (pas brouillon)
- Vérifiez que la bonne année scolaire/trimestre est sélectionnée
- Contactez l'enseignant si les notes manquent

**Montant du paiement incorrect**
- Revoyez le barème des frais avec l'administration
- Vérifiez les exemptions ou réductions appliquées
- Vérifiez l'historique des paiements pour l'exactitude

**Non-réception des notifications**
- Vérifiez que les paramètres de notification sont activés
- Vérifiez les informations de contact email/téléphone
- Consultez les dossiers spam/indésirables

## Bonnes pratiques

1. **Surveillance régulière** : Consultez le portail chaque semaine pour les mises à jour
2. **Communication** : Répondez promptement aux messages
3. **Rester informé** : Lisez régulièrement les annonces
4. **Planification des paiements** : Surveillez le barème des frais et les dates d'échéance
5. **Contacts d'urgence** : Gardez les informations de contact à jour
6. **Sécurité** : Utilisez des mots de passe forts et déconnectez-vous après utilisation
7. **Plusieurs enfants** : Configurez les notifications de manière appropriée pour chaque enfant

## Confidentialité et sécurité

- Votre compte donne accès à des informations sensibles sur les enfants
- Ne partagez jamais vos identifiants de connexion
- Déconnectez-vous des appareils partagés
- Signalez toute activité suspecte immédiatement
- Gardez les informations de contact à jour pour les urgences
