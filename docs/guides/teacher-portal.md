# Guide du Portail Enseignants

## Vue d'ensemble

Le Portail Enseignants NovaConnect permet aux enseignants de gérer les classes, enregistrer l'assiduité, saisir les notes, communiquer avec les élèves et les parents, et accéder aux ressources pédagogiques.

## Prise en main

### Connexion

1. Accédez à l'URL NovaConnect de votre école
2. Entrez votre email et votre mot de passe
3. Cliquez sur "Se connecter"

### Configuration initiale

Lors de la première connexion :
1. Vérifiez vos informations de profil
2. Configurez vos préférences de notification
3. Consultez les classes et matières assignées
4. Familiarisez-vous avec les listes de classe

## Tableau de bord

Le tableau de bord enseignant fournit :
- **Emploi du temps du jour** : Classes du jour
- **Tâches en attente** : Assiduité non prise, notes à publier
- **Messages récents** : Des élèves, parents, administration
- **Événements à venir** : Examens, réunions, échéances
- **Actions rapides** : Tâches courantes d'un coup d'œil

## Gestion des classes

### Consultation des classes assignées

1. Naviguez vers **Classes**
2. Consultez toutes les classes assignées par :
   - Matière
   - Niveau
   - Campus/salle
3. Cliquez sur une classe pour les détails

### Liste de classe

1. Sélectionnez une classe dans **Classes**
2. Consultez l'onglet **Élèves**
3. Voyez les élèves inscrits avec :
   - Photos des élèves
   - Informations de contact
   - Dates d'inscription
   - Notes spéciales

### Prise de l'assiduité

1. Naviguez vers **Classes > [Sélectionner une classe]**
2. Cliquez sur l'onglet **Assiduité**
3. Sélectionnez la date et la période
4. Marquez chaque élève :
   - ✓ Présent
   - ✗ Absent
   - ◐ Retard
   - ⚠ Excusé
5. Ajoutez des remarques si nécessaire
6. Cliquez sur **Enregistrer**

**Bonne pratique** : Prenez l'assiduité au début de chaque classe

### Assiduité groupée

Pour plusieurs périodes :
1. Naviguez vers **Assiduité > Assiduité rapide**
2. Sélectionnez la date
3. Prenez l'assiduité pour toutes les classes séquentiellement
4. Enregistrez tout en une seule fois

## Gestion des notes

### Saisie des notes

1. Naviguez vers **Classes > [Sélectionner une classe] > Notes**
2. Sélectionnez le type d'évaluation :
   - Devoirs quotidiens
   - Tests/Quiz
   - Examens
   - Projets
3. Cliquez sur **Ajouter une note**
4. Remplissez :
   - Nom/description de la note
   - Date
   - Points maximum
   - Scores des élèves
5. Enregistrez comme **Brouillon** ou **Publier**

**Brouillon vs Publié** :
- **Brouillon** : Visible uniquement des enseignants, modifiable
- **Publié** : Visible des élèves et parents, nécessite l'approbation de l'administration pour modifier

### Publication des notes

1. Naviguez vers les notes de la classe
2. Consultez les brouillons de notes
3. Cliquez sur **Publier** lorsque prêt
4. Ajoutez des remarques de publication si nécessaire
5. Confirmez la publication

**Note** : La publication des notes envoie des notifications aux élèves et parents

### Saisie groupée de notes

1. Naviguez vers **Classes > [Sélectionner une classe] > Notes**
2. Cliquez sur **Importation groupée**
3. Téléchargez le modèle
4. Remplissez les notes hors ligne
5. Téléversez le fichier rempli
6. Mappez les colonnes et importez

### Analyse des notes

1. Naviguez vers **Classes > [Sélectionner une classe] > Analyse**
2. Consultez :
   - Moyenne de classe
   - Distribution des notes
   - Taux de réussite/échec
   - Comparaisons entre les trimestres
   - Tendances de performance des élèves

## Performance des élèves

### Vue individuelle d'un élève

1. Naviguez vers **Élèves**
2. Recherchez ou sélectionnez un élève
3. Consultez :
   - Profil et informations de contact
   - Historique des notes
   - Dossier d'assiduité
   - Tendances de performance
   - Contacts des parents

### Rapports de progression

Générez des rapports de progression des élèves :
1. Naviguez vers **Élèves > [Sélectionner un élève]**
2. Cliquez sur **Générer un rapport**
3. Sélectionnez le type de rapport et la période
4. Prévisualisez et téléchargez le PDF

## Communication

### Messagerie avec les élèves

1. Naviguez vers **Messages > Composer**
2. Sélectionnez le(s) destinataire(s) :
   - Élève individuel
   - Classe entière
   - Plusieurs élèves
3. Rédigez le message
4. Envoyez immédiatement ou planifiez

### Messagerie avec les parents

1. Naviguez vers **Messages > Composer**
2. Sélectionnez le(s) destinataire(s) parent(s)
3. Ajoutez l'objet et le message
4. Envoyez avec l'option de copier l'élève

### Annonces

Envoyez des annonces à toute la classe :
1. Naviguez vers **Annonces**
2. Sélectionnez la classe (ou toutes les classes)
3. Rédigez l'annonce
4. Choisissez le mode de diffusion :
   - Notification du portail
   - Email
   - SMS
5. Envoyez immédiatement ou planifiez

### Réunions parents-professeurs

1. Naviguez vers **Réunions**
2. Consultez les réunions planifiées
3. Demandez une réunion avec le(s) parent(s)
4. Définissez l'ordre du jour et les notes
5. Envoyez des rappels de réunion

## Ressources pédagogiques

### Plans de cours

1. Naviguez vers **Ressources > Plans de cours**
2. Créez un nouveau plan de cours
3. Ajoutez :
   - Objectifs
   - Matériels
   - Activités
   - Évaluations
4. Joignez des fichiers
5. Partagez avec d'autres enseignants si nécessaire

### Partage de documents

Partagez du matériel avec les élèves :
1. Naviguez vers **Ressources > Matériel de classe**
2. Sélectionnez la classe
3. Téléversez des documents :
   - Documents distribués
   - Présentations
   - Vidéos
   - Liens
4. Organisez par sujet/date

### Gestion des devoirs

1. Naviguez vers **Devoirs**
2. Créez un devoir :
   - Titre et description
   - Date d'échéance
   - Points maximum
   - Joindre les instructions
3. Assignez à la/aux classe(s)
4. Suivez les soumissions
5. Notez et fournissez un feedback

## Gestion des examens

### Création d'examens

1. Naviguez vers **Examens > Créer un examen**
2. Remplissez les détails de l'examen :
   - Matière
   - Date/heure
   - Durée
   - Attribution des salles
   - Surveillants
3. Ajoutez les instructions de l'examen
4. Publiez l'horaire des examens

### Saisie des notes d'examen

1. Naviguez vers **Examens > [Sélectionner un examen]**
2. Saisissez les scores des élèves
3. Ajoutez des remarques si nécessaire
4. Publiez les notes une fois approuvées

## Rapports et analyses

### Rapports de performance de classe

1. Naviguez vers **Rapports > Performance de classe**
2. Sélectionnez la classe et la période
3. Générez un rapport avec :
   - Scores moyens
   - Distribution des notes
   - Corrélation avec l'assiduité
   - Tendances d'amélioration

### Rapports de notes

1. Naviguez vers **Rapports > Rapports de notes**
2. Sélectionnez les critères :
   - Classe
   - Matière
   - Période
3. Générez et exportez (PDF, Excel)

### Rapports d'assiduité

1. Naviguez vers **Rapports > Assiduité**
2. Sélectionnez la plage de dates et la classe
3. Générez un rapport montrant :
   - Pourcentages d'assiduité
   - Modèles d'absentéisme
   - Élèves problématiques

## Bonnes pratiques

### Assiduité
- Prenez l'assiduité au début de chaque classe
- Ajoutez des remarques pour les absences
- Suivez les absences chroniques
- Informez les parents des préoccupations

### Notation
- Saisissez les notes promptement après les évaluations
- Fournissez un feedback significatif
- Gardez les brouillons de notes jusqu'à vérification
- Publiez les notes de manière cohérente

### Communication
- Répondez aux messages dans les 24 heures
- Communiquez tôt les problèmes de performance
- Documentez les conversations importantes
- Utilisez un ton professionnel

### Gestion des données
- Enregistrez régulièrement votre travail
- Gardez une sauvegarde des données importantes
- Vérifiez l'exactitude des données des élèves
- Signalez promptement les problèmes techniques

## Dépannage

### Problèmes courants

**Classe n'apparaît pas**
- Vérifiez l'assignation avec l'administration
- Vérifiez l'année scolaire en cours
- Assurez-vous que la matière est correctement assignée

**Impossible de publier les notes**
- Vérifiez si les notes sont en statut brouillon
- Vérifiez que l'évaluation est complète
- Contactez l'administration pour les permissions de publication

**Élève pas dans la liste de classe**
- Confirmez l'inscription de l'élève
- Vérifiez avec l'administration de l'école
- Vérifiez la bonne classe/section

**Messages ne s'envoient pas**
- Vérifiez les informations de contact du destinataire
- Vérifiez les paramètres de notification
- Essayez une méthode de contact alternative

## Politiques et lignes directrices

### Délais de saisie des notes
- Devoirs quotidiens : Dans les 3 jours
- Tests/Quiz : Dans les 5 jours
- Examens : Dans les 7 jours
- Notes finales : Selon le calendrier académique

### Politiques d'assiduité
- Prenez l'assiduité pour chaque classe
- Marquez les élèves absents après 15 minutes
- Documentez les absences excusées
- Signalez les préoccupations d'assiduité à l'administration

### Normes de communication
- Ton professionnel dans tous les messages
- Temps de réponse dans les 24-48 heures
- Documentez les communications importantes
- Respectez la confidentialité et la vie privée

## Rappels de sécurité

- Ne partagez jamais vos identifiants de connexion
- Verrouillez votre ordinateur lorsque non utilisé
- Déconnectez-vous après chaque session
- Protégez la confidentialité des données des élèves
- Signalez immédiatement les incidents de sécurité
