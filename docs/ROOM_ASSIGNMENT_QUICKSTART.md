# 🚀 Démarrage Rapide - Attribution des Salles

> **5 minutes pour activer l'automatisation**

---

## ⚡ En 3 étapes

### 1️⃣ Prérequis (2 min)
Créez dans l'ordre :
```
✓ Salles       → Ressources → Salles
✓ Classes      → Classes → Nouvelle
✓ Professeurs  → Utilisateurs → Professeurs
✓ Matières     → Académique → Matières
```

### 2️⃣ Activer (30 sec)
```
Paramètres → Attribution des salles → Activer
```
- ✅ Module activé
- ⬜ Auto-publication (désactivé au début)
- ✅ Notifications activées

### 3️⃣ Créer un emploi du temps (2 min)
```
Planning → Emploi du temps → Nouveau cours
```
Remplissez : Classe, Matière, Professeur, Jour, Heure  
⚠️ **Laissez le champ "Salle" VIDE !**

---

## ✅ C'est tout !

**À 18h00 aujourd'hui**, le système attribuera automatiquement les salles pour demain.

**Vérifier demain matin :**
```
Planning → Attributions de salles
```

---

## 📱 Notifications automatiques

| Quand | Qui reçoit | Message |
|-------|------------|---------|
| 1h avant | Prof + Étudiants | "Cours dans 1h - Salle X" |
| 15min avant | Prof + Étudiants | "Cours imminent - Salle X" |

---

## 🎯 Prochaines étapes

Après 3-4 jours de test :
1. Si satisfait → Activer "Auto-publication"
2. Sinon → Ajuster manuellement les salles
