# Guide d'utilisation NovaConnect PWA

Ce guide explique comment installer et utiliser l'application NovaConnect en mode PWA (Progressive Web App) sur votre ordinateur ou mobile.

## Qu'est-ce qu'une PWA ?

Une PWA est une application web qui peut être installée sur votre appareil et utilisée comme une application native, avec des fonctionnalités avancées :

- ✅ **Installation directe** depuis le navigateur
- ✅ **Fonctionnement hors ligne** (mode avion, zones blanches)
- ✅ **Notifications** pour les événements importants
- ✅ **Mise à jour automatique** en arrière-plan
- ✅ **Interface fluide** comme une application native

## Installation de l'application

### Sur Chrome (Android, Windows, Mac)

1. **Ouvrez** NovaConnect dans Chrome
2. **Attendez** que la bannière d'installation apparaisse en haut de l'écran
3. **Cliquez** sur "Installer" ou l'icône ➕ dans la barre d'adresse
4. **Confirmez** l'installation
5. **L'application** apparaît maintenant dans votre menu Applications

### Sur Safari (iOS)

1. **Ouvrez** NovaConnect dans Safari
2. **Tapez** sur le bouton Partager ()
3. **Faites défiler** vers le bas et tapez sur "Sur l'écran d'accueil"
4. **Personnalisez** le nom si vous le souhaitez
5. **Tapez** sur "Ajouter"
6. **L'icône** NovaConnect apparaît sur votre écran d'accueil

### Sur Firefox (Android, Windows, Mac)

1. **Ouvrez** NovaConnect dans Firefox
2. **Cliquez** sur l'icône d'installation dans la barre d'adresse
3. **Confirmez** l'installation
4. **L'application** est installée sur votre appareil

### Sur Edge (Windows, Mac, Android)

1. **Ouvrez** NovaConnect dans Edge
2. **Cliquez** sur l'icône d'application dans la barre d'adresse (³˙)
3. **Sélectionnez** "Installer cette application en tant qu'application"
4. **Confirmez** l'installation

## Utilisation hors ligne

L'application NovaConnect fonctionne même sans connexion internet !

### Ce qui fonctionne hors ligne

✅ **Consultation de l'emploi du temps**
- Voir vos cours de la semaine
- Accéder aux horaires et salles

✅ **Affichage des notes**
- Consulter les notes déjà téléchargées
- Voir les relevés de notes

✅ **Accès au cahier de texte**
- Lire les devoirs et leçons
- Voir les documents déjà chargés

✅ **Marquage de la présence**
- Scanner les codes QR
- Marquer les présences (synchronisées plus tard)

### Comment utiliser le mode hors ligne

1. **Lancez** l'application normalement
2. **Un indicateur** "Mode hors ligne" apparaît en haut de l'écran
3. **Utilisez** l'application normalement
4. **Vos actions** sont automatiquement enregistrées
5. **Dès que** vous reconnectez à internet, tout se synchronise

### Synchronisation automatique

🔄 **Reconnexion** automatique : Quand vous retrouvez internet, l'application synchronise automatiquement toutes les données.

📊 **Indicateur de synchronisation** : Vous pouvez voir combien d'opérations sont en attente de synchronisation en haut de l'écran.

## Les notifications

### Activer les notifications

Pour recevoir les notifications importantes :

1. **Ouvrez** l'application NovaConnect
2. **Allez** dans Paramètres ()
3. **Sélectionnez** "Notifications"
4. **Activez** les notifications que vous souhaitez recevoir

### Types de notifications

- 🔔 **Présence** : Rappels de marquage de présence
- 📝 **Devoirs** : Nouveaux devoirs à faire
- 📊 **Notes** : Nouvelles notes publiées
- 📅 **Emploi du temps** : Changements d'horaire
- 💰 **Paiements** : Factures et échéances

### Gérer les notifications

Vous pouvez personnaliser les notifications selon vos préférences :

- ✅ Poussées (apparaissent même si l'application est fermée)
- ✅ Dans l'application (uniquement quand l'app est ouverte)
- ✅ Par email (en plus des notifications push)

## Mise à jour de l'application

L'application se met à jour automatiquement en arrière-plan !

### Quand une mise à jour est disponible

1. **Une notification** apparaît en haut de l'écran
2. **Cliquez** sur "Mettre à jour"
3. **L'application** redémarre avec la nouvelle version
4. **Vos données** sont préservées automatiquement

### Mise à jour manuelle

Pour vérifier les mises à jour manuellement :

1. **Tirez** vers le bas sur l'écran d'accueil
2. **Ou** allez dans Paramètres → À propos
3. **Tapez** sur "Vérifier les mises à jour"

## Fonctionnalités avancées

### Raccourcis d'application

Sur certains appareils, vous pouvez ajouter des raccourcis :

- 📸 **Scanner présence** : Accès direct au scan QR code
- 📊 **Notes** : Voir rapidement les notes
- 📅 **EDT** : Accéder à l'emploi du temps
- 🔔 **Notifications** : Voir les dernières notifications

### Partage vers NovaConnect

Vous pouvez partager des fichiers directement vers l'application :

1. **Ouvrez** un fichier (PDF, image, document)
2. **Tapez** sur Partager
3. **Sélectionnez** NovaConnect
4. **Le fichier** est ajouté à l'application

### Écran de démarrage

L'application s'affiche instantanément avec son propre écran de démarrage, sans attendre le chargement du navigateur.

## Dépannage

### L'application ne s'installe pas

**Vérifiez** :
- ✅ Utilisez-vous un navigateur compatible ? (Chrome, Safari, Firefox, Edge)
- ✅ Avez-vous assez d'espace de stockage ?
- ✅ L'adresse URL commence-t-elle par `https://` ?

### L'application ne fonctionne pas hors ligne

**Première utilisation** :
- ✅ Connectez-vous à internet
- ✅ Ouvrez l'application une première fois
- ✅ Laissez les données se charger
- ✅ Vous pouvez maintenant l'utiliser hors ligne

### Les notifications ne s'affichent pas

**Vérifiez** :
- ✅ Les notifications sont activées dans les paramètres de l'app
- ✅ Les notifications du navigateur sont autorisées
- ✅ Le mode "Ne pas déranger" est désactivé
- ✅ Les paramètres système autorisent les notifications pour NovaConnect

### La synchronisation ne fonctionne pas

**Essayez** :
- ✅ Vérifiez votre connexion internet
- ✅ Fermez et rouvrez l'application
- ✅ Tapez sur l'indicateur de synchronisation
- ✅ Attendez quelques secondes après la reconnexion

### L'application est lente

**Conseils** :
- ✅ Videz le cache des données inutiles
- ✅ Ferrez les autres applications en arrière-plan
- ✅ Vérifiez votre connexion internet
- ✅ Mettez à jour l'application

## Sécurité et confidentialité

### Vos données sont sécurisées

- 🔐 **Connexion sécurisée** par HTTPS
- 🔑 **Authentification** via Supabase
- 🛡️ **Données locales** chiffrées sur votre appareil
- 🚫 **Aucune donnée** partagée sans votre consentement

### Supprimer l'application

Pour désinstaller l'application :

**Android** : Appui long → Désinstaller

**iOS** : Appui long → Supprimer l'application

**Windows/Mac** : Paramètres → Applications → Désinstaller

Vos données cloud restent disponibles sur votre compte !

## Foire aux questions (FAQ)

**Q : Puis-je utiliser l'application sur plusieurs appareils ?**
R : Oui ! Installez-la sur tous vos appareils avec le même compte.

**Q : Combien de place l'application prend-elle ?**
R : Environ 50-100 Mo selon votre utilisation et les données téléchargées.

**Q : Dois-je payer pour utiliser la PWA ?**
R : Non, l'application est gratuite. Seules les fonctionnalités de l'école peuvent être payantes.

**Q : Puis-je utiliser l'application sans compte Google/Apple ?**
R : Oui, utilisez simplement votre compte NovaConnect.

**Q : L'application fonctionne-t-elle sur les tablettes ?**
R : Oui, elle est optimisée pour tablettes iPad et Android.

**Q : À quelle fréquence l'application se met-elle à jour ?**
R : Les mises à jour sont automatiques et silencieuses environ une fois par semaine.
